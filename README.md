# RECRUITMENT-MANAGEMENT-SYSTEM
Recruitment Management System is a web-based application that automates the hiring process. It allows users to browse jobs and apply with resumes, while admins can manage applications, select or reject candidates, and send email notifications. Built using Flask, MongoDB, HTML, CSS, and JavaScript.

## Overview

The system allows job seekers to browse job listings, search for relevant roles, and apply by submitting their details along with resumes. On the other side, the admin can manage applications, review candidate profiles, and make decisions such as selecting or rejecting candidates.

This project demonstrates a complete end-to-end recruitment workflow using modern web technologies.

## Features

- Browse and search job listings  
- Apply for jobs with resume upload  
- Admin panel to manage applications  
- Select or reject candidates  
- Automated email notifications  
- MongoDB database integration  
- User-friendly interface  

## Technologies Used

Backend: Python (Flask)  
Frontend: HTML, CSS, JavaScript  
Database: MongoDB  
Email Service: SMTP (Gmail)

## Installation and Setup

1. Clone the repository  
git clone https://github.com/CMRAKESH/RECRUITMENT-MANAGEMENT-SYSTEM.git  

2. Navigate to the project folder  
cd recruitment-management-system  

3. Install dependencies  
pip install flask pymongo  

4. Run the application  
python app.py  

5. Open in browser  
Running on http://127.0.0.1:5000
